# Completed Plans

Completed plans are archived here so agents can use them as historical context
without confusing them for active work.

## Structure

- `architecture/` - completed architecture and refactor plans.
- `audit/` - completed audit plans and verification plans.
- `features/` - completed feature-specific plans.
- `plans/` - completed general implementation, migration, prompt, and handoff plans.
- `refactor/` - completed refactor and maintainability plans.

`docs/ongoing/` is intentionally empty unless new work is explicitly opened.
