# Kie.ai Magic-Wand Research Ledger

Status: active source ledger
Updated: 2026-06-24

Purpose: source-backed prompt guidance for FlashBoard magic-wand profiles. Keep
this in `docs/ongoing/` until the related prompt profiles and service entries
ship, then archive with the implementation plan.
